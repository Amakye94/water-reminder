Water Reminder

A personal automation project: a Rust CLI tool to track daily water intake (log, status, reset) with local file storage.

GitHub Repository

Features

  • Log water intake in milliliters
  • Check daily progress against a goal
  • Reset the daily log
  • Offline, fast, and simple

Build & Run

git clone https://github.com/Amakye94/water-reminder.git
cd water-reminder
cargo build --release

Usage

water-reminder log 250
water-reminder status
water-reminder reset

Created for a personal automation assignment (Rust + CLI + Linux).
